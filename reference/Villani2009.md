# Villani 2009 data

Swedish macroeconomic data

## Usage

``` r
Villani2009
```

## Format

### `Villani2009`

A data frame with 123 rows and 7 columns:

- country:

  Country name

- iso2, iso3:

  2 & 3 letter ISO country codes

- year:

  Year

## Source

Villani, M. (2009). Steady-state priors for vector autoregressions.
*Journal of Applied Econometrics*, 24(4), 630-650.
